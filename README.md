# 헬스케어 물리치료 클리닉 웹사이트 🏥

물리치료 따라하기 운동 프로그램을 제공하는 반응형 웹사이트입니다.

## ✨ 주요 기능

### 🔄 무한 스크롤
- **IntersectionObserver API** 사용
- 처음 8개 프로그램 로드
- 스크롤 시 자동으로 8개씩 추가 로드
- 20개 프로그램이 무한 반복

### 🎯 따라하기 운동 프로그램
총 20가지 운동 프로그램:
- 목 스트레칭 (거북목 개선)
- 어깨 회전 운동 (오십견 예방)
- 허리 강화 운동 (요통 완화)
- 무릎 강화 운동
- 골반 교정 운동
- 코어 근력 운동
- 전신 스트레칭
- 척추 교정 운동
- 그 외 다양한 부위별 운동

### 📋 프로그램 정보
각 프로그램 카드에는 다음 정보가 표시됩니다:
- **운동 시간**: 5분 ~ 25분
- **난이도**: 초급 🟢 / 중급 🟠 / 고급 🔴
- **카테고리**: 목/어깨, 허리, 무릎, 골반, 코어 등
- **조회수**: 실시간 인기도 표시
- **평점**: 4.6 ~ 5.0 별점
- **인기 뱃지**: 🔥 인기 프로그램 표시

### 💙 즐겨찾기 기능
- 하트 버튼으로 관심 프로그램 저장
- 로컬스토리지에 저장되어 재방문 시 유지

### 📱 반응형 디자인
- **데스크톱**: 그리드 레이아웃 (3-4열)
- **태블릿**: 2-3열 그리드
- **모바일**: 1열 세로 레이아웃

### 📅 예약 시스템
- 이름, 연락처, 희망 날짜, 치료 종류 입력
- 간단한 예약 접수 기능

## 🚀 사용 방법

1. `index.html` 파일을 브라우저에서 엽니다
2. 스크롤을 내려 다양한 운동 프로그램을 확인하세요
3. 하트 버튼으로 관심 프로그램을 즐겨찾기에 추가하세요
4. 프로그램 카드를 클릭하면 상세 정보를 볼 수 있습니다
